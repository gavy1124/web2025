19. 웹서버 운영하기

Live Server
웹브라우저 = 제품군? 제품명?
= 제품군. (Apache, IIS, Nginx 등등등)


첫번째 : 브라우저 >> 파일 직접열었을때
두번째 : 웹서버 >>> 브라우저 >>> 파일 열었을때


127.0.0.1 자기 컴퓨터를 가리키는 약속 (localhost)
혼자쓰는 거라 예시일뿐 쓸모는없음;;
하나의 컴퓨터에 여러개의 서버가 동작하고 있을경우를 대비해, 서버마다 포트번호가 있음.


실제 웹서버 사용 하는 경우를 표현.
1. Web Browser >> Web Server >> index.html 순으로
2. index.html >>> Web Server >> Web Browser
어느 나라 웹브라우저에서도 접근 가능토록 하는 순서 표현
